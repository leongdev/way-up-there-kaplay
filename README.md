# Way Up There - Typescript + Kaplay + Tauri

![alt text](docs/image.png) 

A typescript game made with [KAPLAY](https://kaplayjs.com/) and ported to Desktop using [TAURI 2.0](https://v2.tauri.app/)

# How to run

```
bun install
```

```
bun run dev
```
